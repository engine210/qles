# qles - Quick Linux Environment Setup Script

## Quick Start
```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/engine210/qles/master/main.sh)"
```

## content
* zsh
  * Change shell to zsh
  * Install oh-my-zsh
  * Change oh-my-zsh setting and theme
* vim
  * Change vim setting
  * Change vim theme
* tmux
  * Change tmux setting

* git
  * setup my name and email

* ssh
  * import my ssh config file

* ssh config
  * private?

## Requirement
* bash
* curl or wget
* git
